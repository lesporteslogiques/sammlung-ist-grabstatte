# Sammlung ist Grabstätte

« Collections are tombs »

![screenshot](sammlung_ist_grabstatte_screenshot_titlescreen.png)

Enter the museum's storage areas and free the native spirits trapped within the stolen masks preserved in the collections.

Azerty keymappings : ZQSD to move (on a french keyboard), mouse to move view, SPACE to jump (if you think that can help).

Game made for the 2026 Global Game Jam.

https://globalgamejam.org/games/2026/sammlung-ist-grabstatte-5
